Волшебная палочка Гарри Поттера

Описание товара

Настоящая волшебная палочка Гарри Поттера.
Подойдет для любого человека, который решил переквалифицироваться в волшебника прямо сегодня!
Палочка сделана из остролиста с сердцевиной из пера феникса, её сделал мастер Олливандер.

Стоимость

конфет - 18кг

Характеристики

  • Материал — остролист
  • Cердцевина — перо феникса
  • Длина — 28 см
  • Три режима — для новичков, опытный, эксперт
  • wi-fi — 802.11 a/b/g/n/ac/ax 2.4G+5 ГГц, HE80, MIMO, 1024-QAM
  • Версия Bluetooth — Bluetooth v5.0

Подробное описание товара

Палочка продается в связи с выходом на пенсию предыдущего владельца, полного переосмысления всей филисофии бытия и полного ухода в ирл.
Настоящая палочка будет передана настоящему любителю приключений с добрыми помыслами, которые подтверждаются соответствующей справкой из ПНД.
Палочка продается в футляре, футляр в утке, утка в зайце — при отправке Почтой России будет крайне удобно, не сломается.
